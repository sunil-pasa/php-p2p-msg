🔄 How the P2P System Works

The P2P Network application uses a hybrid architecture that combines:

  • Client-Server model for authentication and data storage
  • WebSocket for real-time messaging
  • WebRTC for direct peer-to-peer file transfer

💡 What Happens During P2P Connection

  1. User clicks "P2P" button - Initiates WebRTC connection
  2. Browser creates RTCPeerConnection - With STUN servers
  3. Creates SDP Offer - Session Description Protocol
  4. Sends offer via WebSocket - To the other peer
  5. Other peer receives offer - Creates answer
  6. Exchange ICE candidates - Network paths
  7. Direct P2P connection established - Data flows directly
