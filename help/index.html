Welcome to P2P Network Documentation

This is a complete Peer-to-Peer (P2P) web application built with PHP that enables real-time communication between users without requiring a central server for data transfer.

🎯 Key Features

  • User Authentication - Secure registration and login with JWT tokens
  • Real-time Messaging - Instant messaging via WebSocket
  • P2P Connections - Direct peer-to-peer connections using WebRTC
  • File Transfer - Share files directly between peers
  • Peer Discovery - Find and connect with other users

⚠️ Important Notes

  • You need PHP sockets extension enabled for WebSocket to work
  • Database must be created before running the application
  • Two servers must be running: WebSocket and HTTP
