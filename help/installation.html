🚀 Complete Installation Guide

📋 Prerequisites

  • XAMPP (or any Apache/MySQL/PHP stack)
  • PHP 8.2+ with sockets extension
  • MySQL 5.7+
  • Web browser with WebRTC support (Chrome, Firefox, Edge)

1Enable PHP Sockets Extension

  1. Open php.ini file: C:\xampp\php\php.ini
  2. Find this line: ;extension=sockets
  3. Remove the semicolon (;) to enable it:
;extension=sockets
 Change to:
extension=sockets
⚠️ After enabling, you may need to restart Apache or any running PHP processes.

2Setup MySQL Database

  1. Open phpMyAdmin: http://localhost/phpmyadmin
  2. Click on "New Database" to create a new database
  3. Enter database name: p2p_network
  4. Select collation: utf8mb4_unicode_ci
  5. Click "Create"
  6. Click on the newly created database
  7. Go to "Import" tab
  8. Select the file: config/database.sql
  9. Click "Go" to import
✅ Database is ready with sample users (admin, alice, bob) - all with password: password

3Configure Database Credentials

  1. Open config/config.php
  2. Update database credentials:
'database' => [
    'driver' => 'mysql',
    'host' => '127.0.0.1',
    'port' => 3306,
    'database' => 'p2p_network',
    'username' => 'root',        // Your MySQL username
    'password' => '',            // Your MySQL password
    'charset' => 'utf8mb4',
],

4Create Required Directories

Create these empty folders in the project root:

  • logs/ - For application logs
  • uploads/ - For file uploads

5Start WebSocket Server

Open a terminal and run:

c:\xampp\php\php.exe c:\xampp\htdocs\test\kilo\p2p\server\websocket.php
Expected output:
P2P WebSocket Server started on ws://127.0.0.1:8081
Waiting for connections...
⚠️ Keep this terminal open! The WebSocket server must keep running.

6Start PHP Development Server

Open a NEW terminal and run:

c:\xampp\php\php.exe -S localhost:8080 -t c:\xampp\htdocs\test\kilo\p2p\public

Or use XAMPP Apache:

  1. Open XAMPP Control Panel
  2. Click "Config" next to Apache
  3. Select "httpd.conf"
  4. Add a virtual host or set DocumentRoot to c:\xampp\htdocs\test\kilo\p2p\public
  5. Start Apache

7Access the Application

Open your web browser and navigate to:

http://localhost:8080
✅ Installation complete! You should see the login page.

🧪 Testing the Application

How to Test

  1. Login with sample accounts:
    • Username: admin, Password: password
    • Username: alice, Password: password
    • Username: bob, Password: password
  2. Open a second browser window (or incognito mode)
  3. Login with a different account
  4. Connect with the other user
  5. Send messages

🔧 Troubleshooting

Error Solution
socket_create() not found Enable PHP sockets extension in php.ini
Database connection error Check database credentials in config/config.php
WebSocket connection failed Make sure WebSocket server is running on port 8081
Messages not sending Check browser console (F12) for JavaScript errors
P2P connection failed Use HTTPS in production; check STUN server settings
