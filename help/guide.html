📖 How to Use the P2P Network Application

Quick Overview

This guide explains how to use the P2P Network application for messaging and file sharing with other users.

1Login or Register

Option A: Use Existing Account

If you imported the database, you can use these test accounts:

Username Password
admin password
alice password
bob password

Option B: Register New Account

  1. Click the "Register" tab
  2. Enter a username (e.g., "john")
  3. Enter your email (e.g., "john@example.com")
  4. Enter a password (at least 6 characters)
  5. Click "Register"

2Understanding the Interface

After logging in, you'll see the main interface:

Section Description
Sidebar Left panel with search, peers list, and conversations
Peers Tab Shows all users you can connect with
Messages Tab Shows your conversation history
Chat Area Right panel where messages appear

Status Indicators

  • ● Green - User is online
  • ● Grey - User is offline

3Connect with Other Users

  1. Make sure you're logged in with one account (e.g., "alice")
  2. Open a new browser window or incognito tab
  3. Login with a different account (e.g., "bob")
  4. In the "Peers" tab, find the other user
  5. Click the "Connect" button next to their name
  6. The other user will see a connection request
  7. Once connected, you can start chatting!
⚠️ Note: Both users need to be connected to the WebSocket server for real-time messaging.

4Send Messages

Starting a Chat

  1. Click on a user's name in the Peers list
  2. The chat interface will open on the right
  3. Type your message in the text box
  4. Press Enter or click "Send"

Message Features

  • Real-time delivery - Messages appear instantly
  • Typing indicator - See when the other person is typing
  • Message history - Previous messages are saved
  • Read receipts - Messages marked as read automatically

5Establish P2P Connection

The P2P button creates a direct peer-to-peer connection for file sharing.

How to Use P2P:

  1. Open a chat with a connected user
  2. Click the "🔗 P2P" button
  3. Wait for the connection to establish
  4. You'll see "P2P Connected!" when ready

✅ Benefits of P2P Connection:

  • Direct file transfer without server
  • Faster for large files
  • Reduced server load
  • More secure (data doesn't go through server)

6Share Files

Sharing a File:

  1. First, establish a P2P connection (see step 5)
  2. Click the 📎 (attach) button
  3. Select a file from your computer
  4. The file will be transferred directly to the other user
  5. Progress will be shown
  6. Once complete, the other user can download the file
File Size Transfer Method
Small (<16KB) Sent as single message
Large (>16KB) Chunked transfer with progress
⚠️ Maximum file size: 100MB

7Manage Connections

Connection Status Types:

Status Description
None Not connected - click "Connect" to send request
Pending Waiting for the other user to accept
Accepted Connected - can chat and share files

8Logout

To logout:

  1. Click the logout button (🚪) in the top-right corner
  2. You'll be returned to the login page
💡 Tip: Your messages are saved even after you logout. They will be there when you login again.

🧪 Testing Tips

Best Way to Test

  1. Open two browser windows - One for each user
  2. Use different browsers - Chrome and Firefox work well
  3. Or use incognito mode - For separate sessions
  4. Check the WebSocket server terminal - You'll see connection messages

❓ Frequently Asked Questions

Question Answer
Why aren't messages sending? Check that both users are online and connected to WebSocket
P2P button is greyed out You must first connect with the user via the Connect button
File transfer failed Make sure P2P connection is established first
Can't see other users Make sure you're logged in and check the Peers tab
Online status not updating Refresh the page - this updates status from the server
